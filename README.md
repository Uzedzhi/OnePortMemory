# Однопортовая память в verilog

RTL-описание памяти с поддержкой одновременного чтения и записи. Есть два варианта программы соответственно в ветках "Version 1" и "Version 2". Вторая версия без использования переменной величины в квадратных скобках.
Критический путь будет при выполнении операции чтения. Для чтения необходимо записать в регистр значение ram[addr]. Если предположить, 
что DEPTH = 32, то понадобится мультиплексор, 32 в 1, критический путь которого(если выбирать иерархическую структуру) - это 5 мультиплексоров 2 в 1. То есть
O(log(DEPTH)).
